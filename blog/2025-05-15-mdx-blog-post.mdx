## 오늘 배운 것: CI/CD 파이프라인 구성의 기본

DevOps의 핵심 중 하나는 **CI/CD 자동화**입니다.  
오늘은 GitHub Actions를 사용하여 기본 CI 파이프라인을 구성하는 법을 배웠습니다.

```yaml title=".github/workflows/ci.yml"
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Set up Java
        uses: actions/setup-java@v4
        with:
          distribution: "temurin"
          java-version: "17"
      - name: Build with Gradle
        run: ./gradlew build
```
